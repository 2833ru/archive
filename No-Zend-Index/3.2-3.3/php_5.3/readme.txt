---------
ВНИМАНИЕ!
---------

Для работы PHPShop Enterprise 3.2-3.3 на серверах с PHP 5.3 и выше тебуется лицензия нового образца версии 2.1 с наличием сериного ключа в лицензии.
Получить такой файл (лицензию 2.1) можно бесплатно, отправив запрос на mail@phpshop.ru с просьбой выдать лицензию нового образца для работы ПО PHPShop 2.1 на серверах с PHP 5.3

Лицензии для новой версии PHPShop Enterprise 3.5 бесплатно не выдаются и требуют наличие активной технической поддержки (оплачивается отдельно).
